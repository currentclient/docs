## App Installation Guide

### Download Locations
- **iOS:** Available on the Apple App Store
- **Android:** Available on Google Play Store
- **Requirements:** iOS 12+ or Android 8.0+
- **Storage:** Approximately 50MB of device storage

## Permission Requirements

<Accordion title="Required Permissions">
  **Phone Permissions:**
  - Make and manage phone calls
  - Access to microphone for voice calls
  - Access to speaker and audio controls
  
  **Notification Permissions:**
  - Display incoming call notifications
  - Show message and activity alerts
  - Background app refresh capabilities
  
  **Contact Permissions:**
  - Sync with CurrentClient contact database
  - Access device contacts for integration (optional)
</Accordion>

<Accordion title="Optional Permissions">
  **Camera and Photos:**
  - Add photos to contact profiles
  - Share images in conversations
  
  **Location Services:**
  - Location-based features and analytics
  - Time zone detection for scheduling
</Accordion>

| Feature | Mobile App | Desktop | Notes |
|---------|------------|---------|-------|
| Make Calls | ✅ | ✅ | Full calling functionality on both |
| Receive Calls | ✅ | ✅ | Push notifications on mobile |
| Inbox Management | ✅ | ✅ | Touch-optimized interface on mobile |
| Contact Management | ✅ | ✅ | Quick add/edit on mobile |
| Broadcasting | ⚠️ | ✅ | Limited broadcast features on mobile |
| Reporting | ⚠️ | ✅ | Basic reporting on mobile |
| Integrations | ⚠️ | ✅ | Basic sync features on mobile |

<Note>
  **Mobile First:** CurrentClient's mobile app is designed for essential communication tasks while on the go. For advanced features like detailed reporting and complex broadcast campaigns, use the desktop version.
</Note>
