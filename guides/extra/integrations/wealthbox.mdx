## Integration Workflow Examples

### New Client Onboarding
1. **Client Added in Wealthbox** → Automatically imports to CurrentClient
2. **Welcome Message Sent** → Logs as activity in Wealthbox
3. **Response Received** → Creates follow-up task in Wealthbox
4. **Meeting Scheduled** → Updates both systems' calendars

### Regular Client Communication
1. **Market Update Broadcast** → Sends to Wealthbox client segments
2. **Client Responses** → Log to individual Wealthbox client records
3. **Follow-up Calls** → Create call activities in Wealthbox
4. **Meeting Notes** → Sync to Wealthbox client files

## Troubleshooting Common Issues

<Accordion title="Sync Problems">
  **Contact Sync Issues:**
  - Verify Wealthbox API permissions are properly configured
  - Check for duplicate detection rules causing conflicts
  - Ensure custom fields are mapped correctly
  - Review integration logs for specific error messages
</Accordion>

<Accordion title="Authentication Problems">
  **Connection Issues:**
  - Re-authenticate the Wealthbox connection
  - Verify your Wealthbox account has API access enabled
  - Check that integration user has appropriate permissions
  - Contact support if authentication continues to fail
</Accordion>

<Tip>
  **Pro Tip:** Use Wealthbox tags and categories strategically in CurrentClient segments to create highly targeted communication campaigns for different client types and investment strategies.
</Tip>
