## Troubleshooting Common Issues

<Accordion title="Integration Not Syncing">
  **Check These Items:**
  - API credentials are still valid
  - Both systems are online and accessible
  - No recent changes to field mappings
  - Integration hasn't reached usage limits
</Accordion>

<Accordion title="Duplicate Records">
  **Solutions:**
  - Review duplicate detection rules
  - Check field mapping configuration  
  - Consider using unique identifiers
  - Clean up existing duplicates before re-syncing
</Accordion>

<Tip>
  **Pro Tip:** Test integrations with a small subset of data first to identify and resolve any issues before syncing your entire database.
</Tip>
