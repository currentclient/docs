## Contact Upload Requirements

<Accordion title="File Format Guidelines">
  - **Phone Numbers:** Use E.164 format (+1234567890) for best results
  - **Email Addresses:** Must be valid email format (user@domain.com)
  - **Names:** Separate first and last names into different columns
  - **Duplicates:** Remove duplicate entries before uploading
  - **File Size:** Maximum file size is 10MB per upload
</Accordion>

## Upload Best Practices

<Steps>
  <Step title="Prepare Your Data">
    Clean and format your contact data before uploading
  </Step>
  <Step title="Test with Small Batch">
    Upload 5-10 contacts first to verify formatting
  </Step>
  <Step title="Map Fields Correctly">
    Ensure contact fields are mapped to the right CurrentClient fields
  </Step>
  <Step title="Review and Verify">
    Check uploaded contacts for accuracy and completeness

<Warning>
  **Important:** Always backup your contact data before performing bulk operations. Uploaded contacts cannot be easily undone if errors occur.
</Warning>

## Common Upload Issues

<Accordion title="Phone Number Format Errors">
  **Problem:** Phone numbers not importing correctly
  
  **Solution:** 
  - Use consistent formatting (+1234567890)
  - Remove special characters like dashes or parentheses
  - Include country code for international numbers
</Accordion>

<Accordion title="Duplicate Contact Detection">
  **Problem:** Contacts being marked as duplicates
  
  **Solution:**
  - Review duplicate detection rules in settings
  - Check for slight variations in names or numbers
  - Use the merge duplicate feature after upload
</Accordion>

<Accordion title="Missing Contact Information">
  **Problem:** Some contact fields not importing
  
  **Solution:**
  - Verify column headers match expected field names
  - Check for hidden characters or extra spaces
  - Ensure data is in the correct format for each field type
</Accordion>

## Data Sources

You can import contacts from various sources:

- **CRM Systems** - Export from Salesforce, HubSpot, etc.
- **Email Platforms** - Gmail, Outlook contact exports
- **Spreadsheets** - Excel, Google Sheets files
- **Other Platforms** - Social media, lead generation tools
- **Direct Entry** - Manual contact addition for small lists

<Tip>
  **Pro Tip:** Use tags during import to automatically categorize contacts by source, campaign, or other criteria for better organization.
</Tip>
