# Changelog

Welcome to the CurrentClient changelog. Here you'll find detailed information about new features, improvements, bug fixes, and other changes in each release.

## Latest Updates

### Version 2.1.0 - January 2025

#### New Features
- Enhanced contact management with advanced filtering options
- Improved mobile app performance and stability
- New integration options for popular CRM platforms

#### Improvements
- Faster loading times across all dashboards
- Better error handling and user feedback
- Updated user interface with improved accessibility

#### Bug Fixes
- Fixed issue with contact synchronization
- Resolved notification delivery problems
- Corrected timezone display issues

---

### Version 2.0.5 - December 2024

#### Improvements
- Enhanced broadcasting campaign analytics
- Improved video creation workflow
- Better mobile app offline functionality

#### Bug Fixes
- Fixed contact import validation
- Resolved messaging delivery delays
- Corrected profile settings save issues

### Version 2.0.0 - November 2024

#### Major Release
- Complete redesign of the user interface
- New advanced broadcasting features
- Enhanced CRM integration capabilities
- Improved mobile app with new features

- Updated API endpoints (see migration guide)
- Changed notification settings structure
- Revised user permission system
